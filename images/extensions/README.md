# Extension thumbnails

Place raster thumbnails here (recommended **16∶9**, e.g. **1920×1080** or **1280×720**) and update `extensions.html` if you change filenames.

| Slug            | Used for     | Suggested filename   |
|-----------------|--------------|----------------------|
| `feed-blocker`  | Feed Blocker | `feed-blocker.png`   |

The page currently references `feed-blocker.svg` as a lightweight placeholder; you can swap the `src` to a `.png` / `.webp` when your artwork is ready.
